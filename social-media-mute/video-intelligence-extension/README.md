# Video Intelligence - 视频智能笔记浏览器插件

## 🎯 功能特性

- ✅ **自动提取金句** - AI 识别视频中的关键观点
- ✅ **智能生成摘要** - 3-5 句话总结核心内容
- ✅ **实时分析** - 边看边记录，无需手动操作
- ✅ **支持双平台** - YouTube 和 B站
- ✅ **本地 AI** - 使用 Ollama，隐私安全
- ✅ **导出功能** - Markdown 格式，可导入 Obsidian

---

## 📦 安装方法

### 1. 确保 Ollama 运行中
```bash
# 检查 Ollama 状态
curl http://localhost:11434/api/tags

# 如果没有运行，启动它
brew services start ollama
```

### 2. 加载扩展到 Chrome

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的 **"开发者模式"**
4. 点击 **"加载已解压的扩展程序"**
5. 选择文件夹：`/Users/miaoyoyo/Metasite/video-intelligence-extension`
6. 完成！扩展图标会出现在工具栏

## 🚀 使用方法

### 基本流程

1. **打开视频**
   - 访问 YouTube 或 B站 视频页面
   - 扩展会自动激活

2. **查看侧边栏**
   - 右侧会自动出现紫色渐变侧边栏
   - 显示 "正在分析视频内容..."

3. **等待分析**
   - AI 会自动提取字幕
   - 生成金句和摘要（约 5-10 秒）

4. **导出笔记**
   - 点击 "📋 导出笔记" 下载 Markdown 文件
   - 或点击 "📄 复制全部" 复制到剪贴板

### 示例场景

**学习技术教程**：
1. 打开 YouTube 技术视频
2. 边看边自动记录关键点
3. 导出笔记到 Obsidian
4. 后续复习时快速回顾

**浏览 B站 内容**：
1. 打开 B站 视频
2. 即使三心二意看，也能获取核心内容
3. 通过金句了解视频精华

## 🔧 故障排除

### 问题 1: 侧边栏没有出现

**可能原因**：
- 视频没有字幕
- Ollama 未运行

**解决方案**：
```bash
# 1. 检查 Ollama
curl http://localhost:11434/api/tags

# 2. 启动 Ollama
brew services start ollama

# 3. 刷新视频页面
```

### 问题 2: 分析失败

**检查扩展控制台**：
1. 右键点击扩展图标
2. 选择 "检查弹出窗口"
3. 查看 Console 日志

### 问题 3: 字幕提取失败

**YouTube**：
- 确保视频有英文或中文字幕
- 尝试手动开启字幕

**B站**：
- 确保视频有 CC 字幕
- 部分视频可能没有字幕数据
